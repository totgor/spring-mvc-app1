Корень проекта main страница!

1. Создан каталог config и классы: SpringConfig и SpringMVCDispatcherServletInitializer.
2. SpringConfig наследует WebMvcConfigurer.
3. В SpringConfig создаются бины: которые подключают шаблоны Thymeleaf, указывают где искать шаблоны страниц html и где искать классы для бинов.
4. SpringMVCDispatcherServletInitializer наследует AbstractAnnotationConfigDispatcherServletInitializer.
5. В методе getServletConfigClasses указывается класс SpringConfig для конфигурирования mvc проекта
6. В методе getServletMappings указывается что все запросы обрабатывает диспечтер сервлетов.
